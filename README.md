# 🚀 Md Raihan Rimon — Developer Portfolio

A modern, 3D, fully-responsive personal portfolio built with **Next.js**, **Three.js**, and **Framer Motion**.

### 🔗 Live Demo → **[raihan-rimon-portfolio.vercel.app](https://raihan-rimon-portfolio.vercel.app)**

---

## ✨ Features

- 🌌 **Interactive 3D hero** — animated distorted blobs, torus knot & sparkles (react-three-fiber)
- 🪄 **3D tilt cards** and scroll-reveal animations with Framer Motion
- 📱 **Fully responsive** — optimized for mobile, tablet & desktop
- 🎨 Glassmorphism UI with gradient glows and animated skill marquee
- 📄 **Downloadable CV** and direct links to email, phone & socials
- ⚡ Static-generated and deployed on **Vercel**

## 🛠️ Tech Stack

| Layer | Technologies |
|-------|--------------|
| **Framework** | Next.js 16 (App Router), React 19, TypeScript |
| **Styling** | Tailwind CSS v4 |
| **3D / Animation** | Three.js, @react-three/fiber, @react-three/drei, Framer Motion |
| **Deployment** | Vercel |

## 🚦 Getting Started

```bash
# Install dependencies
npm install

# Run the dev server
npm run dev

# Build for production
npm run build
```

Open [http://localhost:3000](http://localhost:3000) to view it locally.
